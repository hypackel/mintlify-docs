## Grid Template Areas

Grid template areas let you create layouts by naming grid sections:

```css
.grid-container {
  display: grid;
  grid-template-areas: 
    "header header"
    "sidebar main"
    "footer footer";
  grid-template-columns: 200px 1fr;
  gap: 20px;
}

.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }
```

## Responsive Grid

Use `repeat()` and `minmax()` for responsive layouts:

```css
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 20px;
}
```

## Quick Tips

1. Use grid-template-areas for complex layouts
2. Empty cells can be marked with a dot (.)
3. auto-fit makes layouts responsive
4. minmax() prevents columns from getting too small
