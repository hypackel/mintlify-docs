## What is Flexbox?

Flexbox is a CSS layout model that makes it easier to design flexible responsive layouts. Think of it like a way to arrange items in a container, either in a row or a column.

## Creating a Flex Container

To start using Flexbox, we need to make a container "flexible" by setting `display: flex`.

```html
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
```

```css
.container {
  display: flex;
  /* This makes it a flex container */
}
```

## Main Axis vs Cross Axis

In Flexbox, we have two important axes:
- Main Axis: The primary direction (horizontal by default)
- Cross Axis: The perpendicular direction (vertical by default)

## justify-content Property

The `justify-content` property aligns items along the main axis. Here are the main values:

```css
.container {
  display: flex;
  justify-content: center; /* Centers items horizontally */
}
```

Common values include:
- `flex-start` (default): Items at the start
- `center`: Items at the center
- `flex-end`: Items at the end
- `space-between`: Equal space between items
- `space-around`: Equal space around items

## Quick Tips

1. Always set `display: flex` on the container, not the items
2. The default direction is row (horizontal)
3. justify-content works along the main axis
4. Flexbox is one-dimensional (either row OR column)
