## Basic Data Types

JavaScript has several basic data types that we use to store different kinds of information:

### Strings (Text)
```javascript
let firstName = "John";
let lastName = 'Doe';
let message = `Hello there`;  // Template literal
```

### Numbers
```javascript
let age = 25;        // Integer
let price = 19.99;   // Decimal
let temperature = -5; // Negative number
```

## Basic Math Operations

JavaScript can perform various mathematical operations:

```javascript
let a = 10;
let b = 5;

console.log(a + b);  // Addition: 15
console.log(a - b);  // Subtraction: 5
console.log(a * b);  // Multiplication: 50
console.log(a / b);  // Division: 2
```

## Type Checking

Use `typeof` to check what type of data you're working with:

```javascript
let name = "John";
let age = 25;

console.log(typeof name);  // "string"
console.log(typeof age);   // "number"
```

## Quick Tips

1. Strings must be wrapped in quotes
2. Numbers don't need quotes
3. Use template literals (backticks) for easier string formatting
4. Math operations between numbers and strings might not work as expected

## Common Mistakes to Avoid

1. Adding numbers as strings ('2' + '2' = '22')
2. Forgetting to convert string input to numbers
3. Using mathematical operations with undefined variables
4. Not checking data types before operations
