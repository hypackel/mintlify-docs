## What is JavaScript?

JavaScript is a programming language that makes websites interactive. While HTML provides structure and CSS handles styling, JavaScript adds behavior and functionality to web pages.

## Adding JavaScript to HTML

Since we're working with a single-file editor, we'll use the `<script>` tag to add JavaScript directly in our HTML:

```html
<script>
  // Your JavaScript code goes here
</script>
```

## Console Output

The console is a tool for developers to output messages and debug code. To see the console, press F12 in your browser and click on "Console".

```html
<script>
  console.log('Hello, World!');  // Outputs: Hello, World!
  console.log(42);              // Outputs: 42
</script>
```

## Variables

Variables are containers for storing data. We use `let` to declare variables that might change and `const` for values that won't change.

```javascript
let name = "John";          // A string variable
let age = 25;              // A number variable
const DAYS_IN_WEEK = 7;    // A constant value
```

## Quick Tips

1. JavaScript is case-sensitive (`name` and `Name` are different variables)
2. Always declare variables with `let` or `const`
3. Use meaningful variable names
4. End each statement with a semicolon
5. Use the console to check your outputs

## Common Mistakes to Avoid

1. Forgetting to use quotes around text
2. Using variables before declaring them
3. Misspelling variable names
4. Forgetting to open the console to see outputs
