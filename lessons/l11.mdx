## What is CSS Grid?

CSS Grid is a two-dimensional layout system that lets you create complex layouts with both rows and columns. Unlike Flexbox which works in one direction, Grid works in two directions simultaneously.

## Creating a Grid Container

To start using Grid, we set `display: grid` on the container:

```html
<div class="grid-container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
</div>
```

```css
.grid-container {
  display: grid;
}
```

## Defining Columns

Use `grid-template-columns` to define column sizes:

```css
.grid-container {
  display: grid;
  grid-template-columns: 200px 200px 200px; /* Creates 3 equal columns */
}
```

## Grid Gaps

Add space between grid items using the `gap` property:

```css
.grid-container {
  display: grid;
  grid-template-columns: 200px 200px;
  gap: 20px; /* Adds 20px space between all items */
}
```

## The fr Unit

The `fr` unit represents a fraction of the available space:

```css
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr; /* First column takes 1/3, second takes 2/3 */
}
```

## Quick Tips

1. Grid container only affects direct children
2. fr units are great for responsive designs
3. Use gap instead of margins for spacing
4. Grid lines start at 1, not 0
